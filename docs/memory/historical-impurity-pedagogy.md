**The reframe (load-bearing):** the originally-named tramp axes
(hardenability/Jominy, CCT, residual stress) are the *weakest* for our verified
engines; the measured archaeometallurgy data actually clusters on **toughness /
embrittlement / inclusion-cracking** (ferritic bloomery / phosphoric / wrought
iron) — so the literature **corroborates** the verified-vs-plausible split,
doesn't overturn it.

**The pedagogical spine = P is the signed-impurity FOIL to back-end §5b grain
refinement.** §5b teaches refinement = the lone strength-AND-toughness
co-improver because it acts on ONE variable (boundary area) helping both. P is
the clean inverse: strength **↑** (solid-solution lattice) but toughness **↓**
(GB segregation chemistry) — TWO mechanisms pulling opposite. That contrast is
*why* refinement is special. Natural placement = a teaching cell adjacent to §5b.

**The two axes are NOT equally pinned (the honesty that survived advisor twice):**
- Strength: one clean cited number **+237 MPa per 1 at% P** (Thiele–Hošek;
  ferrite ≈142 HV). Triad-candidate (Maynier/Pickering property-correlation class).
- Toughness: real, well-attested, but **slope UNPINNED** ~**40–70 °C DBTT per
  0.1 wt% P** (7 °C/0.01% → 70 vs another source 40, ~1.75× apart). `grain.py`
  §5b ITT carries **no P term** today. *Real effect, NOT a calibrated coefficient.*
- **Unit-basis trap (advisor catch):** strength is per **at% P**, toughness per
  **wt% P**; 1 at% P ≈ 0.56 wt% P (~5.6× apart) — convert before comparing or
  pinning either number.

**Triad gradient (depth ≠ overselling):** cleanest *new* clearers = (1) P
solid-solution strengthening, (2) **Mn:S → MnS stoichiometry** (S+Mn→MnS
conservation-clean; target Mn:S ≳ 2; = Mushet's Mn that made **Bessemer** work —
history free). Real-but-unpinned = P→DBTT slope. **Reuse-not-rebuild** = D
hardness-as-inverse-proxy (Thiele reads HV→P; existing forward engine backward)
+ E Scheil→P-banding→Stead's/Klemm ghost-lines→provenance (existing F4 Scheil).
History-as-purity-ramp (Bessemer-acid-can't-remove-P → Thomas-basic → BOF→N →
ladle→S) = the motivation for the planned **F2 L_P-vs-basicity** curve.
Red-short (S, FeS ~988 °C hot-tear) vs cold-short (P) = a *closing forging
window* (echoes §17 martempering's T-window). The inclusion = the
stress-concentrator that turns §18's sub-critical residual field into a quench
crack (fracture-side coupling).
